Root Cause:

The auth-service is failing because the JWT parser is returning a null subject for specific legacy tokens that were issued without a subject claim. This leads to a NullPointerException when the downstream identity validator attempts to process the returned user object.

Recommended Fix:

Implement null-safe handling in getUsernameFromToken and add a validation fallback for legacy claims.
